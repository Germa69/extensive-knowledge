# Skeleton loading

```
- Để áp dụng được 1 skeleton loading thì phải:
+ Style cho trạng thái khi mà loading nó sẽ có trạng thái như thế nào
+ Kiểm tra xem có bao nhiêu element cần loading
+ Mặc định ::after luôn luôn nằm phía sau thằng ::before
+ Background dưới dạng 1 gradient
+ Khai báo 1 keyframes (tên)

- Trong thực tế sử dụng skeleton loading vào việc khi chờ đợi 1 API trả về
```
